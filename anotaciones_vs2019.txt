*CREACION DE MODELOS
1.-comando par generar los modelos

scaffold-DbContext "Data source=NBK-CBBVT\SQLEXPRESS; Initial Catalog=prueba; user id=sa; password=rootConexion1;" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Models


2.-Agregar "ConnectionStrings"  appsettings.json, se declara la cadena de conexion 

  "ConnectionStrings": {
    "connectionDB": "Data source=NBK-CBBVT\\SQLEXPRESS; Initial Catalog=prueba; user id=sa; password=rootConexion1;"
  },


*INYECCION DE DEPENDENCIAS
3.- Agregar using en Startup.cs

using PruebaCore.Models;   #nombre del proyecto
using Microsoft.EntityFrameworkCore;

4.- Agregar en la clase Startup en el metodo ConfigureServices(IServiceCollection services)
var coonection = Configuration.GetConnectionString("connectionDB");
            services.AddDbContext<pruebaContext>(options => options.UseSqlServer(coonection));
Se declara el contexto para realizar la inyeccion de dependecias.
